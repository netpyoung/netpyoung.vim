## Windows environment variable
* VIM = d:\myHome\.vim

## clone

```bash
git clone git@github.com:netpyoung/netpyoung.vim.git

or 

git clone https://github.com/netpyoung/netpyoung.vim.git
```

## submodule update

```
git submodule init
git submodule update
```
